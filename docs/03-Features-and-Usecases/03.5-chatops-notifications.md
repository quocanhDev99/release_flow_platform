# ChatOps & Notifications

## 1. Feature Overview
The ChatOps feature bridges the Release Flow Platform with team communication tools (Telegram, Email, MS Teams, Slack). It broadcasts real-time alerts whenever a significant event occurs, such as a deployment schedule being created, updated, or an AI OCR import completing. It supports both Global/System-wide broadcasts and Personal Direct Messages (DMs).

## 3. Sequence Diagram (Event-Driven Notification)

```mermaid
sequenceDiagram
    autonumber
    actor User as Release Manager
    participant API as NestJS API
    participant EventBus as Event Emitter
    participant NotifSvc as Notification Service
    participant Config as Settings Config
    participant External as Teams / Slack / Telegram

    User->>API: POST /api/deployments (Create Schedule)
    API->>API: Save to DB
    API->>EventBus: emit('deployment.created', payload)
    
    EventBus->>NotifSvc: Handle Event async
    NotifSvc->>Config: Get active Webhooks and Tokens
    Config-->>NotifSvc: Webhook URLs & Tokens
    
    NotifSvc->>NotifSvc: Build Notification Payloads (Adaptive Cards / Markdown)
    
    par Send to all channels
        NotifSvc->>External: POST /webhook (Slack)
        NotifSvc->>External: POST /webhook (MS Teams)
        NotifSvc->>External: POST /bot{token}/sendMessage (Telegram)
        NotifSvc->>External: SMTP sendMail (Email)
    end
    
    External-->>NotifSvc: HTTP 200 OK
    NotifSvc->>NotifSvc: Log Success
```
