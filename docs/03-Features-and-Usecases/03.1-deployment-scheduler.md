# Deployment Scheduler

## 1. Feature Overview
The Deployment Scheduler allows Release Managers and Developers to visually manage multi-environment deployment schedules on an interactive calendar grid. It provides drag-and-drop capabilities, bulk actions, and real-time conflict detection to ensure deployments are executed smoothly.

## 3. Sequence Diagram (Schedule a Deployment)

```mermaid
sequenceDiagram
    autonumber
    actor User
    participant Frontend as Angular UI
    participant Backend as NestJS API
    participant DB as PostgreSQL (Prisma)
    participant Notifications as Notification Service

    User->>Frontend: Select date and time on Calendar
    Frontend->>Frontend: Open 'New Deployment' Dialog
    User->>Frontend: Fill details (Environment, Release Version) & Submit
    Frontend->>Backend: POST /api/deployments
    
    Backend->>DB: Query existing deployments (Conflict Check)
    DB-->>Backend: Return active overlapping deployments
    
    alt Conflict Detected
        Backend-->>Frontend: 400 Bad Request (Conflict Error)
        Frontend-->>User: Show Error Message
    else No Conflict
        Backend->>DB: INSERT into DeploymentWindow
        DB-->>Backend: Deployment Record created
        Backend->>Notifications: Trigger Schedule Created Event
        Notifications-->>Backend: Success
        Backend-->>Frontend: 201 Created
        Frontend-->>User: Show Success Toast & Refresh Calendar
    end
```
